**Setup Configuration**
- Open config.env and change all environment variable

**Step Running**
1. Import .sql to your local machine
2. Running main.go with command go run main.go

**API ROUTES**
1. /user [POST]
2. /user/{page}/{limit} [GET] 
3. /user/{id} [GET]
4. /user/{id} [PUT]
5. /user/{id} [DELETE]
